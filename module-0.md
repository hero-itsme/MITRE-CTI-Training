# Module 0: Introducing MITRE ATT&CK® for CTI

This module serves as the foundation and roadmap for the full training series, establishing the core definitions and utility of the framework in threat intelligence.

---

## 🎯 1. Core Definition: Cyber Threat Intelligence (CTI)
The training adopts Sergio Caltagirone’s definition of CTI:
* **Definition:** Actionable knowledge about adversaries and their activities.
* **Goal:** Enables defenders to make better security decisions and reduce harm.

## 🏗️ 2. How ATT&CK Enhances CTI
The primary value of the ATT&CK framework is its ability to **structure data**. This allows analysts to:

### **Compare Behaviors**
* Contrast different threat groups.
* Track how a single group evolves over time.
* Compare adversary tactics against existing internal defenses.

### **Provide a Common Language**
* **Analyst to Defender:** Identifying a specific technique (e.g., *T1547.001 - Registry Run Keys*) allows a defender to immediately know which data sources (Registry data) are needed for detection.
* **Community Sharing:** Organizations can share intelligence using standardized IDs that are universally understood.

## 🚀 3. Training Goals
The course outlines five key objectives:
1. **Understand Utility:** Why ATT&CK is the standard for CTI.
2. **Mapping:** How to extract ATT&CK techniques from narrative reports and raw data.
3. **Data Management:** How to store and visualize mapped data.
4. **Analysis:** How to perform threat analysis using the mapped data.
5. **Actionability:** How to turn intelligence into concrete defensive recommendations.

## 🗺️ 4. Roadmap of the Training Series
Module 0 sets the stage for the following lessons:
* **Module 1:** Mapping from Narrative Reporting (External reports).
* **Module 2:** Mapping from Raw Data.
* **Module 3:** Storing and Analyzing (Using tools like ATT&CK Navigator).
* **Module 4:** Making Defensive Recommendations.
